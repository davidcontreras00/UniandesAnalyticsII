Existen en Maching Learning cuatro formas de aprendizaje automático: 
1. Aprendizaje supervisado: Básicamente se trabaja con un modelo predictivo, generado a partir de datos que se tienen a priori etiquetado y clasificado, es decir que ya se tiene identificado a qué grupo, valoro categoría pertenecen los registros. El grupo mencionado es denominado grupo de entrenamiento que suele consistir en pares de objetos en los cuales el input está relacionado con el output, con este grupo se llevan a cabo las estimaciones y ajuste del modelo planteado.
   De acuerdo a lo anterior, de esta forma el algoritmo va aprendiendo a clasificar las muestras comparando el modelo con la etiqueta real de la muestra realizando las compensaciones respectivas al modelo de acuerdo a cada error en la estimación del resultado. Algunos algoritmos que se pueden implementar en este tipo de aprendizaje son: K vecinos más cercanos, redes neuronales artificiales, árboles de decisión y regresión logística. 
2. Aprendizaje no supervisado: Funcionan de manera similar a los supervisados pero en este caso no se tiene información a priori que permita validar la clasificación. El modelo se ajusta a las observaciones que se tienen como input, los datos de entrada no se encuentran clasificados ni etiquetados; además no son necesarias en este tipo de aprendizaje. Son ejemplo de este algoritmo y siendo los más utilizados los modelos de clasificación, ya que forma grupos entre casos/registros que poseen características similares entre sí. Algunos algoritmos que se pueden implementar en este tipo de aprendizaje son: K-medias, clúster jerárquico y mapas auto-organizados.
3. Aprendizaje semisupervisado: Es una combinación de los dos métodos anteriores, se cuenta con información a priori en algunos casos, pero en otros se cuenta sólo con el input. Se busca en este caso la construcción de un modelo supervisado mejorado ya que los datos etiquetados pueden incrementar y se tendría un mejor resultado en el modelo. 
4. Aprendizaje por esfuerzo: Son problemas no supervisados que sólo reciben re-alimentaciones o refuerzos, el sistema aprende a base de ensayo-error. El objetivo es aprender la función de valor que maximice la recompensa de manera tal que se optimice las políticas de la forma como se comporta el mundo que le rodea y a tomar buenas decisiones para el cumplimiento de los objetivos. 
   Los tres métodos donde se usa este tipo de aprendizaje son la programación dinámica, métodos Monte Carlo y aprendizaje de diferencias temporales.
